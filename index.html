Welcome to the HackerYou Summer Academy!

In order to ensure everyone has the right command line tools and programs, we'll all be developing on a Virtual Machine, or VM. The VM is custom to the HackerYou environment and includes Ruby 2.0, Rails 4.0, git, mysql, and some other nice things. You're going to use a tool called Vagrant to run your VM invisibly inside your main operating system. This means you can use your operating system's native programs (like a text editor and web browser) to work on things, but they'll all be run from a command line that is secretly UNIX.

That's a lot of words. Don't worry, it's super simple to set up:

1. Install VirtualBox

VirtualBox is the actual software that will invisibly run our virtual machines. Simply download it and run the installer.

  • Windows: download
  • Mac: download

2. Install Vagrant

Vagrant is the software that manages your virtual machine on your local computer, and makes it easy to run commands in the VM. It's also super simple to set up; just download it and, again, run the installer.

Download

3. Download the HackerYou Vagrantfile

The Vagrantfile is something we've configured with all of the software you'll need in the VM.

  • Click the giant Download .ZIP button at the top of the page.
  • Unzip the downloaded file on your computer.
    • Windows: Double click the file, and press the Extract All toolbar button.
    • Mac: Will either happen automatically, or just double click the file.
  • Rename the newly created and unzipped directory to hacker_you. This will become your main working directory where you'll keep all of your projects!
  • Move the new hacker_you directory into your home folder, so it's easy to get to.
    • Windows: C:/Users/ME
    • Mac: The default folder that opens when you open a Finder window, aka /Users/ME.

4. Run vagrant up

  • Open a command prompt.
    • Windows: Press Start, type cmd, and then press Enter.
    • Mac: Open Applications/Utilities/Terminal
  • Type cd hacker_you and press Enter. (You'll understand what this does in just a minute!)
  • Type vagrant up.

This could take quite a while, so while it's working, you should get started watching Intro to Command Line!

5. Jump into your new VM!

Your virtual machine is now running invisibly in the background. Whenever you want to start working with it (for example, to follow along with the video above or to start working on a project), you'll want to follow very similar steps:

  • Open a command prompt.
  • Type cd hacker_you and press Enter.
  • Type vagrant ssh.

After you type vagrant ssh, you'll see that the command prompt changes to signify that you're running inside the Vagrant VM. You can now safely practice all of the commands you're learning in the screencast or type rails new my_app to start a new Rails app!

What now?

Finish up with the video, and start playing around in your new VM. If you're feeling a bit lost about the command line, don't worry, we'll review it a little bit in class. If you have trouble getting the VM working, don't hesitate to ask a mentor!
